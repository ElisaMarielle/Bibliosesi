# Bibliosesi

## Descrição do projeto
O **Bibliosesi** é uma plataforma digital desenvolvida com o objetivo de modernizar e facilitar o acesso a biblioteca da escola Sesi. A aplicação permite que os estudantes explorem o catálogo de livros de forma prática e intuitiva, visualizando informações como sinopses e avaliações de outros alunos.
Além disso, o sistema oferece funcionalidades para acompanhamento de empréstimos, incluindo prazos de devolução e controle pessoal de livros. O projeto também conta com uma área dedicada a notícias e conteúdos sobre o universo literário, incentivando o interesse e o hábito da leitura entre os usuários.
